# Urban Plant Cookies — Primera Entrega (HTML + CSS)

Este proyecto forma parte de la **Primera Entrega** del Integrador.  
El objetivo de esta etapa es construir la estructura visual completa de un e‑commerce utilizando únicamente HTML semántico, CSS organizado y diseño responsive.

---

## Sobre el proyecto
Urban Plant Cookies es un e‑commerce de venta de cookies con estética New York style, enfocado en productos plant‑based, diseño moderno y una experiencia visual cuidada.

En esta entrega se desarrolló la maqueta estática completa del sitio, priorizando coherencia visual, estructura clara y buenas prácticas de maquetación para preparar el camino hacia las próximas etapas del integrador.

---
## Funcionalidades principales
- Página principal con hero, categorías y productos destacados  
- Listado de productos  
- Detalle de producto  
- Carrito 
- Formulario de registro  
- Formulario de contacto 
- Página “Sobre nosotros” con identidad de marca  
- Página de administración

- **HTML**  
- **CSS**
